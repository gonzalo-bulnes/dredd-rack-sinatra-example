Dredd::Rack Sinatra Example
===========================

Demonstrate the [Dredd::Rack][dredd-rack] usage in a Sinatra app.

  [dredd-rack]: https://github.com/gonzalo-bulnes/dredd-rack

Usage
-----

```bash
# run the app
rackup --port=4567
```

Development
-----------

```bash
# run the app
rerun 'rackup --port=4567'
```

Documentation
-------------
